## 端口不变
- 前端：`8080`
- 后端：`8000`

## 启动方案（不改代码）
- 后端：在 `nocode/python_api/python` 运行 `python -m uvicorn main_api:app --host 0.0.0.0 --port 8000 --reload`
- 前端：在 `nocode` 运行 `npm run dev -- --port 8080`（强制监听 8080，不改配置文件）
- 如 8080 已被占用：先关闭占用该端口的进程（例如之前启动的 Vite），再重启上述命令；仍不改代码。
